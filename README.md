# GUI Brick Breaker Game

A Java-based implementation of the classic arcade game Brick Breaker (also known as Arkanoid or Breakout), built with Java Swing.

The player controls a paddle at the bottom of the screen and must deflect a ball upwards to destroy a wall of bricks. The game is won when all bricks are destroyed. The player loses if the ball falls past the paddle.



## Features

-   **Classic Arcade Gameplay:** Control a paddle to break bricks with a bouncing ball.
-   **Keyboard Controls:** Use the Left and Right Arrow keys to move the paddle.
-   **Score Tracking:** Gain points for every brick you destroy.
-   **Multiple Levels:** The game features a map of bricks that are cleared as you play.
-   **Win/Loss Conditions:** The game ends with a "You Won!" or "Game Over" message.
-   **Restart Functionality:** Press 'Enter' to play again after a game ends.

## How to Play

1.  The game starts automatically.
2.  Use the **Left and Right Arrow keys** to move your paddle.
3.  Keep the ball in play by bouncing it off your paddle.
4.  Destroy all the bricks on the screen to win.
5.  If the ball falls below your paddle, you lose.
6.  When the game is over, press **Enter** to start a new game.

## How to Compile and Run

This project requires the Java Development Kit (JDK) to be installed. It uses only standard Java libraries.

1.  **Save the Code:**
    Save the Java code into a file named `BrickBreaker.java`.

2.  **Open a Terminal or Command Prompt:**
    Navigate to the directory where you saved the file.

3.  **Compile the Code:**
    ```bash
    javac BrickBreaker.java
    ```

4.  **Run the Game:**
    ```bash
    java BrickBreaker
    ```

A new window will appear with the game running. Enjoy breaking some bricks!
